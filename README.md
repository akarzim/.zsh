# .zsh
My ZSH configuration

## How to Install

Clone the repository

```sh
git clone git@github.com:akarzim/.zsh.git ~/git/zsh
```

Then link the dotfiles

```sh
cd ~/git/zsh
ln -s ./zshenv ~/.zshenv
ln -s ./zshrc ~/.zshrc
ln -s ./zprofile ~/.zprofile
ln -s ./zlogin ~/.zlogin
ln -s ./zlogout ~/.zlogout
ln -s ./zplug-packages.zsh ~/.zplug/packages.zsh
```
